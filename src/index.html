SCSS-helper-library

Although this is meant to replace the less-css-helper-library, see less-css-helper-library for how to use the library for now. This web page will be updated soon to contain accurate details and documentation of the scss-helper-library. You can help contribute to that too. All types of input or contributions welcome.

Class name abbreviations

Display styles

b ➾ block display

f ➾ flex display

g ➾ grid display

Margin auto

auto ➾ margin right and left auto, margin top and bottom zero

Container (display) styles

b-container ➾ block display, with margin auto and width set. Suitable as an outer container

Flex (display) styles

c = center

sb = space between

jcc = justify content center
