Collabo styles-library

This is temporary documentation for the styles-library, which will replace the less-css-helper-library. You can help contribute to code and/or documantation too. All types of input or contributions welcome.

Class names, detached rulesets and mixins

Margin auto (detached rulesets and/or mixins only)

  • auto ➾ margin right and left set to auto (margin top and bottom set to zero)

Block display

  • b ➾ block display

Display + margin auto:

  • b-auto ➾ b + auto

Container (display) style, useful as outer container:

  • b-container ➾ b-auto + width

Flex display (detached rulesets and/or mixins only)

  • col ➾ flex-direction set to column
  • sb ➾ justify-content set to space-between
  • jc-center ➾ justify-content set to center
  • ai-center ➾ align-items set to center

Flex display

  • f ➾ flex display
  • f-col ➾ f + col
  • f-sb ➾ f + sb
  • f-center ➾ f + jc-center + ai-center
  • f-jc-center ➾ f + jc-center
  • f-ai-center ➾ f + ai-center

Display + margin auto:

  • f-auto ➾ f + auto
  • f-col-auto ➾ f-col + auto
  • f-sb-auto ➾ f-sb + auto

Container (display) styles, useful as outer containers:

  • f-container ➾ f-auto + width
  • f-container-col ➾ f-col-auto + width

Grid display

  • g ➾ grid display
